# Hold The System!
An interactive web-based game where the user can simulate a 3-body problem. You begin by initializing the positions and masses of three planets across space. Then, you define their velocities and trajectories in a pool-like style. Start the system and watch it unfold.

Your goal is to **HOLD THE SYSTEM**—that is, keep it alive for as long as possible while still introducing chaos into the three-body system. The system ends when all planets drift too far apart or when two planets collide.

Your score is based on how long the system survives and how much the planets interact with each other.
